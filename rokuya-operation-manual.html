はじめに

▾

このマニュアルでは、専門的な知識がなくても運用いただけるよう、画面の場所と操作手順を中心にまとめています。コードの編集は必要ありません。

Point困ったときは、まず「どの場所（管理画面 or テーマエディター）での作業か」を確認すると迷いません。次の章で2つの場所を整理します。
Tip大きな変更を行う前に、現在のテーマを複製しておくと安心です（→ 第10章）。

2つの管理場所と基本用語

▾

サイトを管理する2つの場所

場所主にできること開き方
① Shopify 管理画面 商品・コレクション・ページ・ブログ・ファイル・メニュー・翻訳など「中身（データ）」の管理 admin.shopify.com にログイン
② テーマエディター 各ページの「見た目・配置・文言」の編集、セクションの追加・削除・並び替え オンラインストア → テーマ → カスタマイズ

2

商品の管理

▾

新しい商品を追加する

  1. 管理画面左の 商品管理 → 商品 を開き、右上の 「商品を追加する」 をクリック。
  2. タイトル（商品名）と説明を入力。
  3. メディアに商品画像をアップロード（後からの差し替え・追加も可能です）。
  4. 価格を入力。
  5. 右側のステータスを、準備中は「下書き」、公開する場合は「アクティブ（販売中）」に設定。
  6. 下部のメタフィールド欄でROKUYA独自の情報を設定（次項）。
  7. 右上の「保存」をクリック。
写真が後追いの場合情報を先に登録して公開し、画像は仮（ダミー）または後日差し替え、という運用で問題ありません。画像はいつでも 商品 → メディア から追加・差し替えできます。

ROKUYA独自の設定（メタフィールド）

商品ページ下部のメタフィールドで、以下を設定すると、サイト上の表示や絞り込みに正しく反映されます。

項目役割設定値の例
作家（Artist）商品ページと一覧に作家名を表示作家メタオブジェクトを選択
季節タグ（Season）季節での絞り込み・表示Spring / Summer / Autumn / Winter / Others
一点もの（Inquire only）ONにすると購入ボタンが「お問い合わせ」に変化（→第3章）オン／オフ
完売（Sold）完売表示に切替該当時に設定
産地・技法（Origin / Technique）商品ページに産地や技法の解説を表示テキスト

器種・素材の分類

器種（皿・碗など）と素材（陶器・磁器・ガラスなど）は、対応するコレクションに商品を含めることで分類されます（→ 第4章）。

「購入」と「お問い合わせ」の出し分け

▾

ROKUYAでは、商品によってボタンの表示が自動で切り替わります。

商品の状態表示されるボタン
通常販売（在庫あり）カートに追加／購入
一点もの（メタフィールド「一点もの」がオン）この器について問い合わせる（お問い合わせページへ誘導）
完売（メタフィールド「完売」を設定）Sold / 完売 表示 ＋ お問い合わせ
設定方法一点もの・受注品として扱いたい商品は、商品ページのメタフィールド「一点もの（Inquire only）」をオンにするだけです。これで購入ボタンが自動的にお問い合わせ導線へ切り替わります。

コレクションの管理

▾

新しいコレクションを作成する

  1. 商品管理 → コレクション を開き、「コレクションを作成する」をクリック。
  2. タイトルと説明を入力。
  3. コレクションの種類を選択：
    • 手動…商品を一つずつ追加する
    • 自動…条件（タグ・季節 等）に一致する商品を自動で含める
  4. 画像（任意）を設定し、「保存」。
  5. 必要に応じて、メニューに追加して表示します（→ 第7章）。
Tip季節コレクションは、商品の季節タグを条件にした自動コレクションにすると、新商品を追加するだけで自動的に反映され、運用が楽になります。
3軸の考え方ROKUYAのコレクションは「器種」「素材」「季節」の3軸で整理されています。新規追加時も、この3軸のどこに位置づくかを意識すると、サイト内の絞り込みと一貫します。

ページ・文言の編集（テーマエディター）

▾

テーマエディターの開き方

  1. オンラインストア → テーマ を開く。
  2. 公開中テーマの「カスタマイズ」をクリック。
  3. 上部中央のメニューから、編集したいページ（トップ、各ページ）を選択。

文言（テキスト）を編集する

  1. 左側の一覧、または画面上で編集したいセクションをクリック。
  2. 右側の設定欄でテキストを書き換える。
  3. 右上の「保存」をクリック。

セクションの追加・削除・並び替え

  • 追加：左側の「セクションを追加」から、「ROKUYA」カテゴリーのセクションを選べます。
  • 削除：セクションを選び、設定欄下部の「セクションを削除」。
  • 並び替え：左側一覧でドラッグして順序を変更。
  • 非表示：目のアイコンで、削除せず一時的に非表示にできます。

例：免責事項ページを編集する

「輸出入に関する免責事項」ページは、テーマエディターで本文を自由に編集できます。

  1. テーマエディター上部のメニューで、該当ページを選択。
  2. テキストブロックのセクションをクリック。
  3. 本文（Body）欄で、文章の追記・削除・太字・箇条書き・リンクを編集。
  4. 「保存」。
注意免責事項などの法的な文章は、内容を必ず自社でご確認のうえ運用してください。デザイン枠は用意済みですが、文面の最終責任は運営者にあります。

ファイル（PDF等）の管理

▾

Service Guide などのPDFは、ShopifyにアップロードしてURLを取得し、ボタンに設定します。

PDFをアップロードしてURLを取得

  1. 管理画面左の コンテンツ（Content） → ファイル（Files） を開く。
    （見当たらない場合は、上部の検索で「ファイル」と検索）
  2. 「ファイルをアップロード」からPDFを選択。
  3. アップロード後、ファイル横のリンクコピーアイコンでURLをコピー。

Service Guide のリンクを差し替える

  1. テーマエディターで For Professionals ページを開く。
  2. Resource（Service Guide）セクションをクリック。
  3. PDF link 欄に、コピーしたURLを貼り付けて「保存」。
Pointダウンロードボタンは、Shopifyにアップしたファイルがそのままダウンロードされる仕様です。Google Drive等のリンクは別タブで開くだけになる場合があるため、Content → Files のURLをご利用ください。
7

メニュー（ナビゲーション）の編集

▾

ヘッダーやフッターのリンク（メニュー）は、次の場所で編集します。

  1. コンテンツ（Content） → メニュー（Menus） を開く。
  2. 編集したいメニュー（例：メインメニュー、フッター）を選択。
  3. 「メニュー項目を追加」でリンクを追加（ページ・コレクション・URLなどを指定）。
  4. ドラッグで並び替え、「保存」。
Tip新しいページやコレクション（例：免責事項、新コレクション）を作成したら、ここでメニューに追加すると、来訪者がたどり着けるようになります。
8

Journal（ブログ）の管理

▾

記事を追加する

  1. コンテンツ（Content） → ブログ記事（Blog posts） を開く。
  2. 「ブログ記事を作成」をクリック。
  3. タイトル・本文・アイキャッチ画像を入力。
  4. 右側でブログ（Journal）を選択し、表示状態を「公開」または日時指定。
  5. 「保存」。
9

言語・翻訳

▾

現在、英語・日本語に対応しています。商品やページの翻訳は、ShopifyのTranslate & Adaptアプリで管理します。

  1. 管理画面左の アプリ → Translate & Adapt を開く。
  2. 翻訳対象（商品・コレクション・ページ・テーマ など）を選択。
  3. 各言語の欄に翻訳を入力（自動翻訳の下書き → 手直し、も可能）。
  4. 「保存」。
タイ語についてタイ語対応は、対応範囲・優先度をご相談のうえ、今後追加していく予定です。追加時も同じ Translate & Adapt 上で管理します。

テーマの管理と注意点

▾

変更前にバックアップ（テーマの複製）

  1. オンラインストア → テーマ を開く。
  2. 公開中テーマの「…」メニューから「複製」を選択。
  3. 複製ができたら、本体の編集を行う。問題があれば複製から戻せます。
やらないこと
  • コードエディター（テーマのコード編集）で rokuya- で始まるセクションを削除・改変しない。
  • 公開中テーマで大きな実験をしない（必ず複製で試す）。
  • 独自ドメインのDNS設定（Cloudflare）は、接続済みのため基本的に変更しない。

公開／非公開

準備中のテーマは「非公開」のまま編集し、完成後に「公開」で切り替えます。商品単位では、ステータスを「下書き／アクティブ」で切り替えます。
